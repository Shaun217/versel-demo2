# ⚡️ Speed Click Challenge (光速手速挑战)

这是一个基于原生 HTML5、CSS3 和 JavaScript 开发的网页小游戏。主要用于测试 GitHub 到 Vercel 的自动化部署流程，同时也是一个测试手速的趣味工具。

## 🎮 游戏玩法
1. 点击页面中央的 **“开始游戏”** 按钮。
2. 在 **10秒** 倒计时结束前，疯狂点击按钮！
3. 游戏结束后，系统会根据你的点击次数给出评分。

## 🛠 技术栈
- **HTML5**: 构建游戏结构
- **CSS3**: 现代化 UI 设计，包含动画效果和响应式布局
- **JavaScript**: 处理倒计时、计分逻辑和 DOM 交互

## 🚀 部署状态
本项目已托管至 Vercel，支持自动化构建。
- 只有静态文件，无后端依赖。
- 提交代码到 GitHub 后自动更新。

## 📂 文件结构
- `index.html`: 游戏主界面
- `style.css`: 样式文件
- `script.js`: 游戏逻辑控制
